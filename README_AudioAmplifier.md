# 🔊 Audio Amplifier Using Op-Amp

An analog audio amplifier designed to boost low-level audio signals with **high stability and minimal distortion**, built using an Operational Amplifier (Op-Amp) — covering theoretical analysis, circuit design, simulation, and hardware implementation.

---

## 📌 Project Overview

This project demonstrates the design and implementation of an Op-Amp based audio amplifier capable of amplifying low-level audio signals cleanly and efficiently. The project covers the full engineering workflow from circuit analysis to physical hardware validation.

---

## ⚙️ Circuit Design

| Stage | Component | Function |
|-------|-----------|----------|
| Input Conditioning | Resistors + Capacitors | Signal filtering & coupling |
| Amplification | Op-Amp | Gain control & signal boosting |
| Output Coupling | Capacitor | DC blocking & audio output |
| Power Supply | Decoupling capacitors | Noise reduction & stability |

---

## 🔬 Design & Implementation Steps

1. **Theoretical Analysis** — Performed circuit analysis to determine optimal gain, bandwidth, and component values
2. **Circuit Design** — Designed schematic with input conditioning, gain control, and output coupling stages
3. **Simulation & Validation** — Simulated circuit using **NI Multisim** to verify amplification performance and signal quality
4. **Hardware Implementation** — Built and tested the physical circuit, ensuring stable audio amplification with minimal distortion
5. **Testing & Troubleshooting** — Verified output signal quality and resolved hardware issues during implementation
